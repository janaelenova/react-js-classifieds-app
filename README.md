# React & TypeScript Classifieds Application 🚀

Модерна, брза и оптимизирана веб-апликација за објавување и пребарување на огласи (Oglasnik). Изработена со **React**, **TypeScript** и современ технички стек, апликацијата нуди одлични перформанси, флексибилност и поддршка за повеќе јазици.

---

## 🛠 Технолошки Стек

Апликацијата е изградена со следниве технологии:

* **Frontend:** [React](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
* **Стилизирање:** [Tailwind CSS](https://tailwindcss.com/)
* **База & ORM:** [Prisma](https://www.prisma.io/)

## ✨ Клучни Карактеристики

* **Кориснички Профили:** Регистрација, најава и менаџирање на сопствени огласи.
* **Креирање Огласи:** Напредна форма за додавање огласи со категории, слики и детални описи.
* **Филтрирање и Пребарување:** Брзо филтрирање на огласите по категорија, цена, локација и датум.
* **Мулти-јазична Поддршка (Multi-language):** Интернационализација на интерфејсот за поддршка на повеќе јазици.
* **Респонсивен Дизајн:** Оптимизиран UI/UX за сите уреди (мобилни, таблети и десктоп).

## 🚀 Како да ја стартувате апликацијата локално

Следете ги овие чекори за да го покренете проектот на вашиот локален компјутер:

### 1. Клонирање, инсталација, стартување и седирање на базата
Копирајте го целиот блок команди за одеднаш да го клонирате репозиториумот, да ги инсталирате пакетите, да го покренете серверот и да ја наполните базата:

```bash
# Клонирање на проектот и влегување во папката
git clone [https://github.com/janaelenova/react-js-classifieds-app.git](https://github.com/janaelenova/react-js-classifieds-app.git)
cd react-js-classifieds-app

# Инсталација на сите потребни пакети
npm install

# Стартување на развојниот сервер
npm run dev

# Полнење или комплетно ресетирање на базата со почетни тест-податоци
npx prisma db seed
